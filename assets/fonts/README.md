# OpenDyslexic Fonts

This directory should contain the OpenDyslexic font files for the ClearRead app.

**Required files:**
- `OpenDyslexic-Regular.ttf` — Regular weight font
- `OpenDyslexic-Bold.ttf` — Bold weight font

**Source:** Download from [opendyslexic.org](https://opendyslexic.org/) or the [OpenDyslexic GitHub repository](https://github.com/antijingoist/open-dyslexic).

**License:** SIL Open Font License (OFL) — free for commercial and personal use.

The app will attempt to load these fonts at startup. If they are not present, the app will fall back to system fonts but will display a warning in the console.
